Welcome to mrsd's documentation!
================================

*mrsd* is a Python toolkit to generate MR sequence diagrams, available on `PyPi <https://pypi.org/project/mrsd/>`_. To install from source, the only dependencies are `Matplotlib <https://matplotlib.org/>`_ and `numpy <https://numpy.org/>`_.

To get started, have a look at the tutorials (:doc:`FLASH <flash/index>` and :doc:`multi-echo FLASH <multiecho/index>`) and the `examples <https://github.com/lamyj/mrsd/tree/master/examples>`_. Once familiar with the basic concepts, you can find more details in the :doc:`api/index`.

Rather use MATLAB than Python? `mrisd <https://github.com/benoitberanger/mri_seq_diagram_matlab>`_ is the answer!
